# MarkDown常用语法大全

记录常用的 MarkDown 语法。



## 警告

警告在文档中经常用于引起对警告、注释和提示的注意。Markdown 未提供用于警告的特殊语法，并且大多数 Markdown 应用程序不支持警告（一个例外是 MkDocs）。

但是，如果你需要添加警告，你也许可以使用 引用块 和 表情符号 和 强调 来创建一些类似于你在其他网站上看到的警告。



```
:warning: 警告：警告内容
```

:warning:警告：警告内容



```
:memo: 注释：注释内容
```

:memo: 注释：注释内容



```
:bulb: 提示：提示内容
```

:bulb: 提示：提示内容
